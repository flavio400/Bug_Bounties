# Web3 & Web2 Security & Bug Bounty Portfolio

Welcome to my personal repository dedicated to smart contract security audits, vulnerability research, and bug bounty findings. This space serves as a public portfolio documenting real-world security flaws, deep-dives into protocol architectures, and executable Proof of Concepts (PoCs).

The goal of this repository is to demonstrate a rigorous approach to smart contract auditing, threat modeling, and defense-in-depth engineering.

---

## Methodology & Technical Tooling

Every vulnerability listed here undergoes a rigorous validation pipeline before being documented:

1.  **Static Analysis & Manual Review**: Line-by-line inspection of contract interactions, trust assumptions, and access controls.
2.  **Threat Modeling**: Mapping out privileged roles, user-supplied data paths, and external dependencies (oracles, AMMs).
3.  **Exploit Proof of Concept**: Writing explicit, deterministic test cases utilizing local mainnet forks or comprehensive mocks to simulate production-like conditions.
4.  **Remediation**: Formulating surgical, production-ready code fixes that minimize gas impacts and protocol architecture disruption.
